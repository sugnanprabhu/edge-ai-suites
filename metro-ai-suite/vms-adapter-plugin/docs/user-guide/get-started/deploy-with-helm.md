# Deploy with Helm

This section describes Helm chart deployment for the VMS Adapter Plugin (VAP).

## Status

Helm chart deployment is not available in the current release (Version 1.0.0). Refer to
the [Release Notes](../release-notes.md) for updates on Helm chart availability in future
releases.

## Alternative Deployment

Use Docker Compose to deploy the application. See the [Get Started](../get-started.md) guide
for complete step-by-step instructions. Raise an issue in case of any defects.
