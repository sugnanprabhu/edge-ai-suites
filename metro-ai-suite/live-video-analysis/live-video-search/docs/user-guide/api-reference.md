# API Reference

Live Video Search is composed of Smart NVR and VSS Search microservices. Refer to the corresponding APIs below:

## Core Endpoints

- **VSS Pipeline Manager**: `/manager` endpoints
- **Search‑MS**: `/query`
- **NVR Event Router**: `http://<host-ip>:8000`
