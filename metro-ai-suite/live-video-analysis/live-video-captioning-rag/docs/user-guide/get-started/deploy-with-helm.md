# Deploy with Helm Chart

The Live Video Captioning RAG application is designed as a companion service to Live Video Captioning application and cannot be deployed independently using Helm charts.

For deployment instructions using Helm charts, please refer to the Live Video Captioning [documentation](https://docs.openedgeplatform.intel.com/2026.1/edge-ai-suites/live-video-captioning/get-started/deploy-with-helm.html), which includes guidance on deploying both Live Video Captioning and the Live Video Captioning RAG component together.
