# RVC Requirements

## Knowledge / Experience

- You are familiar with executing Linux commands.
- ROS 2 background strongly recommended.

## Target System

An industrial PC with an Intel® CPU supporting. Internet connectivity is recommended during the initial setup for installing open-source content.

The following table lists the system requirements:

| Attribute | Requirements |
|---|---|
| **Intel Processor** | • 13th Gen Intel® Core™ Processor (Code name: Raptor Lake [**RPL**])<br>• 12th Gen Intel® Core™ Processor (Code name: Alder Lake [**ADL**]) |
| **Memory** | 16 GB RAM (minimum) |
| **Storage** | 16 GB (minimum) |

## Validated Hardware Platforms

The following platforms have been validated with RVC:

- ASRock Industrial intelligent edge platform [(iEPF-9012S-EY4)](https://www.asrockind.com/en-gb/iEPF-9012S-EY4)
- ASRock Industrial intelligent edge platform [(RPL-P)](https://www.asrockind.com/en-gb/iEP-7020E)
